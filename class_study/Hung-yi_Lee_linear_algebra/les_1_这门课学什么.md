# 线性系统特征
1. Persevering Multiplication
```
x -> Linear System -> y

kx -> Linear System -> ky
```
2. Persevering Addition
```
x -> Linear System -> y
a -> Linear System -> v

x+a -> Linear System -> y+v
```

**注意:**  
转置操作是线性的.
![hungyiLee_la_les1_transpose](../../Attachments/hungyiLee_la_les1_transpose.png)

求导操作也是线性的.

$$
f \to f'
$$
$$
kf \to kf'
$$
$$
f+g \to f'+g'
$$

积分也是线性的.
