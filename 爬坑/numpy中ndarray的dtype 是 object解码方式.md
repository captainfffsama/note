# 问题
遇到一个`.npy`文件,`np.load`之后,ndarray中的元素是一个`dict`,其`dtype=object`:
```shell
In [1]: import numpy as np
In [2]: a=np.load("0000092_10207.jpg_0000092_10210.jpg.npy")
In [3]: a                                                                                                                                                                                      
Out[3]:                                                                                                                                                                                        
array({'path1': '0000092_10207.jpg', 'path2': '0000092_10210.jpg', 'matche_pts': [[(26, 142), (35, 141)], [(163, 124), (173, 124)], [(359, 93), (368, 92)], [(534, 105), (546, 105)], [(436, 20
4), (443, 208)], [(335, 169), (340, 169)], [(5, 138), (16, 138)], [(149, 121), (159, 119)], [(114, 167), (121, 166)], [(437, 205), (443, 209)]]},                                              
      dtype=object)   
In [4]: a.shape
Out[4]: ()

```

# 解决方式
使用`a[()]`即可解码:
```shell
In [5]: b=a[()]

In [6]: b
Out[6]: 
{'path1': '0000092_10207.jpg',
 'path2': '0000092_10210.jpg',
 'matche_pts': [[(26, 142), (35, 141)],
  [(163, 124), (173, 124)],
  [(359, 93), (368, 92)],
  [(534, 105), (546, 105)],
  [(436, 204), (443, 208)],
  [(335, 169), (340, 169)],
  [(5, 138), (16, 138)],
  [(149, 121), (159, 119)],
  [(114, 167), (121, 166)],
  [(437, 205), (443, 209)]]}

In [7]: type(b)
Out[7]: dict
```
